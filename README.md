# webpro-express
Latihan membuat Back-end &amp; Front-end dengan module Express pada matakuliah Web Pemrograman

## Project Setup

```sh
npm install
```

```sh
npm install express
```

### Compile and Hot-Reload for Development

```sh
npm run serve 'or' npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
